Bài này thì dùng map sẽ gọn hơn. Ta sẽ nhập vào map rồi chỉ cần cout ra value thôi.

```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    map<int,int> m;
    for(int i=0; i<n; i++) {
        int x; cin >> x;
        m[x]++;
        cout << m[x] << " ";
    }
}
```
